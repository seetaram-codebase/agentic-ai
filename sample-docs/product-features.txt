# Product Features and Roadmap

## Current Features (v1.0)

### Document Management
- **File Upload**: Drag-and-drop interface for PDF and TXT files
- **Automatic Processing**: Documents are automatically chunked and embedded
- **Source Tracking**: Every response includes source citations with page numbers
- **Bulk Upload**: Upload multiple files simultaneously

### Query Interface
- **Natural Language Queries**: Ask questions in plain English
- **Context-Aware Responses**: Answers based on your uploaded documents
- **Source Citations**: See exactly where information comes from
- **Response Streaming**: Real-time response generation (coming soon)

### System Reliability
- **Multi-Region Failover**: Automatic switching between Azure regions
- **Health Monitoring**: Real-time status of all endpoints
- **Error Recovery**: Graceful handling of API failures
- **Rate Limit Handling**: Automatic retry with exponential backoff

### User Interface
- **Electron Desktop App**: Native desktop experience
- **Dark Theme**: Modern, eye-friendly interface
- **Status Indicators**: Visual feedback for system health
- **Responsive Design**: Works on various screen sizes

## Known Limitations

1. Maximum file size: 10MB per document
2. Supported formats: PDF, TXT (more coming soon)
3. Maximum query length: 1000 characters
4. Rate limits depend on Azure OpenAI subscription tier
