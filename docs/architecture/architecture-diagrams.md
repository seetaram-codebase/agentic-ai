## Failover Flow

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         FAILOVER LOGIC                                       │
│                                                                              │
│   ┌─────────┐     ┌──────────────┐     ┌─────────────────────────────────┐  │
│   │ Request │────▶│   Primary    │────▶│  Success? Return Response       │  │
│   └─────────┘     │   Endpoint   │     └─────────────────────────────────┘  │
│                   └──────────────┘                                          │
│                          │                                                   │
│                          │ FAIL (429, 500, Timeout)                         │
│                          ▼                                                   │
│                   ┌──────────────┐     ┌─────────────────────────────────┐  │
│                   │   Secondary  │────▶│  Success? Return Response       │  │
│                   │   Endpoint   │     │  + Mark Primary Unhealthy       │  │
│                   └──────────────┘     └─────────────────────────────────┘  │
│                          │                                                   │
│                          │ FAIL                                              │
│                          ▼                                                   │
│                   ┌──────────────────────────────────────────────────────┐  │
│                   │   Return Error: "All endpoints unavailable"          │  │
│                   └──────────────────────────────────────────────────────┘  │
│                                                                              │
│   RECOVERY: After 60 seconds, retry unhealthy endpoints                     │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```
